Run it directly with Python

TermQuarium ships as an example inside cozy_tui's own repo. It runs anywhere cozy_tui does — Windows, Linux, and macOS terminals alike. No packaged installer for those platforms yet, but nothing stops you from cloning and running it.

shell
git clone https://github.com/youssefahmed2017/cozy_tui.git cd cozy_tui pip install -e . python examples/aquarium/aquarium.py

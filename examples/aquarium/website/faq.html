Questions

Frequently asked.

Straight answers, not marketing.

Is it free?

Yes. The Windows installer and the source are both free.

Does it work on Linux or macOS?

Yes, from source — cozy_tui, the library TermQuarium is built on, has a real POSIX terminal backend alongside its Windows one. There's just no packaged installer for those platforms yet, so you'd run python examples/aquarium/aquarium.py directly. See Download.

Do I need Python installed for the Windows installer?

No. The installer is a standalone packaged executable — Python isn't required.

Can fish die?

Yes. A fish that goes long enough without eating loses health from starvation. Sharks are the other cause — see below.

Can sharks eat the other fish?

Yes. The Shark is a genuine predator — it hunts the smaller fish instead of chasing food, so buying one changes how the rest of the tank behaves.

Do fish have babies?

Yes. Each day, every mutual, grown-up, non-predator pair with a strong enough bond has a chance to breed, up to a sane cap on total fish so it doesn't run away forever.

What happens if I run out of money?

Emergency Aquarium Welfare quietly steps in — a small fresh start, not a game over.

Can I pause?

Yes, Esc — and it actually freezes the simulation, not just the screen, so nothing ages, starves, or breeds while you're away from the keyboard.

How does saving work?

Name a save once and every later save writes back into that same file. The load screen can rename, duplicate, or delete any save. See Saves in the docs.

Wait, are there axolotls now?

Yes. It's buyable from the Shop like any other species, isn't a predator, and can even turn up as a random starter. It's priced in the same range as the fish — the point was never to make it a better pick, just a different kind of resident: it rests far more than any fish, never schools, and has real favorite foods (Brine Shrimp, Bloodworms, Worms). See Fish in the docs.

What's the difference between Fish Food and Treats?

Fish Food is bulk — drop it in the water and whoever's nearest eats it. Treats (Brine Shrimp, Worms, Bloodworms, Plankton, Pizza) are bought from the Shop and fed directly to one chosen fish from its own Inspector, and relieve hunger by exactly the same amount — a favorite just earns a nicer reaction, never a bigger number.

Do fish actually dream?

Yes. Sleeping fish occasionally dream — click the 💭 next to a sleeping fish for a small looping animated scene. Personality leans which kind (a Greedy fish dreams about food often, not exclusively), and a fish's own recent memories can shape it further, including the rare nightmare, which forces it to wake up scared and go find a Friend to sleep beside. See Mechanics in the docs.

What are Achievements, and can I lose them?

Eleven account-wide milestones, always visible (locked or not) from the Achievements menu. They're not tied to any one save — a New Aquarium or a Load never resets what you've already earned.

What's the Cheat Console?

Press ` for a small typed-command console meant for testing — spawning a fish for free, setting money directly, buying from the Shop without the grind. It's a debug tool, not part of normal play, but every command runs through the same real code the Shop and Inspector already use.

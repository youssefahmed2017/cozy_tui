Saving

Press P to save. The first time, you're asked to name the aquarium; every later P writes back into that same named save automatically, rather than prompting again.

Loading

Press L to open the load screen. From there you can load any save, or Rename, Duplicate, or Delete one without loading it first.

Where saves live

Each save is its own versioned JSON file under ~/.termquarium/saves/, named after the aquarium. Writes are atomic — TermQuarium writes to a temporary file and only replaces the real save once that write finishes, so an interrupted save can't corrupt an existing one. Settings live alongside it in the same ~/.termquarium/ folder.

Pausing

Esc pauses — and it actually freezes the simulation, not just the screen, so nothing ages, gets hungry, or breeds while the pause menu is open.

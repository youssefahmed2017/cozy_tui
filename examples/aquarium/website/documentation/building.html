Building from source

How the actual Windows installer gets built, for anyone who wants to package their own.

Contents Overview Fish Decorations Mechanics Controls Saves Building from source

TermQuarium ships as a PyInstaller-packaged executable wrapped in an Inno Setup installer. Both config files live in examples/aquarium/ inside the Cozy TUI repo — run both commands from there.

shell
pyinstaller TermQuarium.spec # -> dist/TermQuarium.exe iscc TermQuarium.iss # -> Output/TermQuarium-Setup.exe

The second step needs Inno Setup installed for its iscc compiler. The resulting TermQuarium-Setup.exe is exactly what's attached to each GitHub release.

Running the tests

The game's pure logic — steering, hunger, economy, relationships, the save format — is unit-tested independently of any real terminal:

shell
python -m pytest tests/test_aquarium.py tests/test_termquarium_save.py tests/test_termquarium_world.py -q
